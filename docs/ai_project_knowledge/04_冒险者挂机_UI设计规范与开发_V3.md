## 第一部分：UI 视觉风格定稿

### 1.1 最终确认风格

韩式Q版幻想网游UI，彩虹岛风格。核心特征：

- 奶白色实心游戏面板（禁止半透明玻璃）
- 浅蓝清晰描边（2px + 圆角）
- 边缘轻微玻璃高光
- 金色点缀（按钮、标题、重要元素）
- 明亮清新的蓝白配色
- 大圆角（10-16px）
- 柔和渐变、轻微浮雕凸起感
- 真实游戏UI（不是网页后台，不是程序灰盒）

### 1.2 颜色系统

| 用途 | 颜色 | 色值参考 |
|---|---|---|
| 面板底色 | 奶白色 | #F5F0E8 ~ #FAF6F0 |
| 面板描边 | 浅蓝 | #7BA7CC ~ #8FB8D6 |
| 标题栏 | 浅蓝渐变 | #6B9AC4 → #8FB8D6 |
| 按钮主色 | 浅蓝 + 金色点缀 | 蓝底金边 |
| 按钮次色 | 浅灰蓝 | #C5D5E4 |
| 高光边 | 白色 | #FFFFFF 80% |
| 品质色：白 | — | #D0D0D0 |
| 品质色：绿 | — | #6BBA75 |
| 品质色：蓝 | — | #5588CC |
| 品质色：金 | — | #D4A843 |
| 品质色：红 | — | #CC4444 |
| 文字主色 | 深灰蓝 | #2C3E50 |
| 文字次色 | 中灰 | #7F8C8D |

### 1.3 禁止风格

伪透明/棋盘格/格子背景（AI生图必须用纯品红 #FF00FF）、白底/黑底/灰底、网页后台样式、程序灰盒、暗黑写实、厚重金属、儿童贴纸、海报/概念图/UI样机、半透明玻璃面板、多个页面混在一起。

## 第三部分：UI 设计系统规范

### 3.1 色彩规范

| 层级 | 色系 | 用途 |
|---|---|---|
| 主色 | #6B9AC4 浅蓝 | 标题栏、主按钮、进度条填充 |
| 辅色 | #8FB8D6 淡蓝 | 描边、次按钮 |
| 底色 | #F5F0E8 奶白 | 所有面板底板 |
| 强调 | #D4A843 金色 | 重要按钮边框、品质标识 |
| 危险 | #CC4444 红色 | 红装品质、低HP警告 |

### 3.2 圆角规范

| 元素 | 圆角半径 |
|---|---|
| 大面板 | 16px |
| 中面板/卡片 | 12px |
| 按钮 | 8px |
| 格子 | 6px |
| 标签 | 4px |

### 3.3 边框规范

| 用途 | 宽度 | 颜色 |
|---|---|---|
| 面板外框 | 2px | #8FB8D6 |
| 按钮边框 | 1.5px | #7BA7CC |
| 格子边框 | 1px | #B0C8DC |

### 3.4 字体与布局规范

| 层级 | 字号 | 字重 | 颜色 |
|---|---|---|---|
| 标题 | 24px | Bold | #2C3E50 |
| 面板标题 | 18px | Bold | #2C3E50 |
| 正文 | 14px | Regular | #34495E |
| 次要文字 | 12px | Regular | #7F8C8D |
| 数值 | 16px | Bold | #2C3E50 |

- 组件间距：8px | 面板内边距：16px
- 按钮最小：120×36px | 格子：48×48px | 装备槽：56×56px

### 3.5 Godot 可视化落地规范

1. 所有UI组件必须有可视化.tscn场景和对应素材
2. 禁止用纯代码生成UI外观
3. 面板底板使用TextureRect + 九宫格素材
4. 按钮使用TextureButton + 多状态素材
5. 文字使用Label + 项目统一字体
6. 布局使用Control锚点系统，禁止绝对坐标硬编码

### 3.6 统一反向限制（生图用）

```text
不要文字，不要乱码，不要占位信息，不要人物，不要装备数值，
背景必须纯品红 #FF00FF，不要伪透明/棋盘格/格子背景，不要白底，不要黑底，不要灰底，不要展示板，
不要海报，不要概念图，不要UI样机，不要网页后台，
不要程序灰盒，不要暗黑写实，不要厚重金属，不要多个页面混在一起。
```

## 第五部分：原型转 UI 流程

核心原则：**先结构，后风格。**

1. 识别页面类型 → 2. 确定布局结构（顶部/中部/底部/辅助区）→ 3. 提取数据需求 → 4. 生成Godot Control节点层级

标准输出结构：
1. 页面定位 2. 视觉风格 3. 页面布局 4. UI元素清单 5. 图像生成提示词 6. 反向限制 7. Godot拆件建议

每次只做一个页面。默认页面结构：顶部区域（标题栏+关闭按钮+资源状态）→ 中间区域（主要内容）→ 底部区域（操作按钮+分类按钮）→ 辅助区域（Tooltip+装饰件）。
