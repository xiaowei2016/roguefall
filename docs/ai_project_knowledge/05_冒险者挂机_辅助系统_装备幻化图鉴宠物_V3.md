# 冒险者挂机 辅助系统 V3：装备幻化、图鉴、宠物

> 项目名：冒险者挂机 | 引擎：Godot 4.7 | 路径：D:\Projects\roguefall

---

## 第一部分：装备幻化系统

### 1. 幻化核心规则

- 玩家可以将装备外观幻化为已拥有的任何装备外观
- 幻化仅改变外观显示，不影响装备属性
- 整套幻化（一次性覆盖全部部位），不是逐部位幻化
- 幻化外观资源使用真透明PNG，在Godot中可视化制作

### 2. 幻化数据绑定

- 幻化记录绑定：当前幻化目标装备外观ID + 幻化来源列表（玩家已拥有装备的ID集合）
- 幻化操作：选择一件已拥有装备 → 整套外观切换为该装备的完整外观
- 取消幻化：恢复为当前实际装备的原始外观
- 幻化状态持久化到本地存档

### 3. 幻化与属性完全解耦

- 装备属性始终由实际穿戴装备决定
- 幻化切换零属性变动
- 新增装备自动加入幻化来源列表

## 第二部分：图鉴系统

### 4. 图鉴类型

| 图鉴类型 | 功能 |
|---|---|
| 怪物图鉴 | 记录击杀怪物种类，按击杀阶段解锁加成 |
| 掉落图鉴 | 记录获得的装备/道具种类 |
| 消耗品图鉴 | 记录使用的消耗品种类 |
| 装备图鉴 | 记录收集的装备种类 |
| 宠物图鉴 | 记录获得的宠物种类 |

### 5. 图鉴属性加成

- 图鉴系统提供永久性微属性加成
- 怪物图鉴：击杀达到阶段阈值解锁对应加成（如击杀100只slime获得攻击+1）
- 其他图鉴：收集达到数量阈值解锁对应加成

---

## 第三部分：宠物系统

### 6. 宠物携带规则

- 最多同时携带3只宠物
- 1只主宠（显示跟随 + 主属性加成生效）
- 2只辅助宠（不显示跟随 + 副属性加成生效）

### 7. 宠物属性加成

| 宠物槽位 | 属性加成比例 |
|---|---|
| 主宠 | 100%属性加成 |
| 辅助宠1 | 50%属性加成 |
| 辅助宠2 | 30%属性加成 |

### 8. 宠物跟随表现

- 主宠在战斗场景中可视化跟随玩家角色
- 辅助宠不显示
- 宠物与怪物不直接交互（不参与战斗）

---

## 第四部分：硬性约束

- 装备幻化必须在Godot中可视化，禁止纯代码拼外观
- 幻化外观资源必须真透明PNG
- 图鉴数据必须持久化到本地存档
- 宠物跟随使用AnimatedSprite2D，动画资源独立于玩家/怪物
- 所有系统资源统一放在 res://assets/ 对应子目录下
